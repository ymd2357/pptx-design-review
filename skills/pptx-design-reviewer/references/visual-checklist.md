# Visual Inspection Checklist (PPTX)

This checklist covers visual inspection for PPTX slide decks.

---

## 0. Evidence Pack (Before/After)

- [ ] Export slide images for the reviewed range (Before)
- [ ] Export slide images for the proposed fix (After)
- [ ] Keep filenames stable (slide-001.png, slide-002.png, ...) for easy comparison

---

## 1. Layout Verification

### Structural Integrity

- [ ] Title area aligns consistently across slides
- [ ] Content area margins are consistent
- [ ] Slide master elements appear correctly
- [ ] Section headers align with the same grid

### Overlap & Clipping

- [ ] No text boxes overlap unintentionally
- [ ] No text-on-text overlap across overlapping text boxes (characters do not collide)
- [ ] No line overlap inside a text box (line spacing prevents collisions)
- [ ] Text is not clipped inside boxes
- [ ] Images do not overlap text
- [ ] If text overlays imagery, readability is protected (overlay/solid box/contrast)

### Alignment

- [ ] Elements align to a consistent grid
- [ ] Icons and text are vertically aligned
- [ ] Multi-column layouts are evenly spaced

---

## 2. Typography Verification

### Readability

- [ ] Body text is readable at presentation distance
- [ ] Line spacing is comfortable
- [ ] Heading hierarchy is clear

### Consistency

- [ ] Font families are consistent
- [ ] Font weights are consistent
- [ ] Emphasis styles (bold/italic) are used sparingly

### Language Handling

- [ ] Line breaks look natural
- [ ] Long words or URLs do not overflow

---

## 3. Color & Contrast Verification

### Contrast

- [ ] Text contrast is sufficient against backgrounds
- [ ] Color is not the sole indicator of meaning

### Palette Consistency

- [ ] Brand colors are used consistently
- [ ] Accent colors are limited and intentional

---

## 4. Image & Media Verification

### Image Quality

- [ ] Images are sharp at intended display size
- [ ] Aspect ratios are preserved
- [ ] Important content is not cropped out

### Media Handling

- [ ] Embedded media plays correctly in target tool
- [ ] Fallback visuals exist if media cannot play

---

## 5. Slide-to-Slide Consistency

- [ ] Titles follow the same position and size
- [ ] Page numbers or footers (if any) are consistent
- [ ] Repeated components match exactly

---

## 6. Accessibility (Practical)

- [ ] Font sizes are large enough for typical viewing distance
- [ ] Contrast supports low-vision readability
- [ ] Dense text is minimized

---

## 7. Motion (if applicable)

- [ ] Animations are minimal and purposeful
- [ ] Auto-advance is disabled unless required
- [ ] A static export (PDF) is available for motion-sensitive viewers

---

## 8. Output & Delivery

- [ ] Slides look correct in target output (PPTX / PDF)
- [ ] Fonts render as expected on another machine
- [ ] No missing assets or broken links

---

## Priority Matrix

| Priority | Category | Examples |
| ---------- | ---------- | ---------- |
| P0 (Critical) | Readability breaking | Overlap, clipped text, unreadable contrast |
| P1 (High) | Serious clarity issues | Blurry images, inconsistent hierarchy |
| P2 (Medium) | Moderate issues | Misalignment, spacing inconsistencies |
| P3 (Low) | Minor issues | Small color deviations |
